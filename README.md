# Treinamento NTEC (Semana 2)

## O que o projeto faz
Projeto em TypeScript que lê e valida variáveis de ambiente, além de funções utilitárias, com manipulação de arrays e objetos.

## Como rodar localmente
1. Clonar o repositório com `git clone https://github.com/liviafrancar/treinamento-ntec.git`
2. Instalar as dependências com `npm install`
3. Criar o `.env` com as informações do `.env.example` e com os seus valores
4. Rodar o projeto com `npm run dev`

## Variáveis de ambiente necessárias
- `DATABASE_URL` (endereço de banco de dados)
- `API_KEY` (chave de acesso à API)
- `PORT` (porta de rede do servidor)

## Estrutura de pastas
- `src/` (onde está o código fonte em TypeScript)
